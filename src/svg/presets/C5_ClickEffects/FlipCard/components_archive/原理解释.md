## 核心原理（三句话版）

1. **零高容器叠罗汗**：用 `height: 0` 让多层内容重叠在同一位置[1]
2. **scale 翻转伪装3D**：通过 `scale(-1→0→1)` 模拟卡片翻转，在 `scale=0` 时切换正反面[2][3]
3. **opacity 配合时间差**：正面在翻转70%时才显示，避免和背面重叠[4]

***

## 自定义模板代码（带注释）

```html
<!-- ========== 第1层：底层装饰背景（可选） ========== -->
<section style="height: 0;">
  <svg viewBox="0 0 750 850" style="
    display: block;
    width: 100%;
    background: url('你的背景图.png') 0 0 / 100% 100% no-repeat;
    pointer-events: none;
  "></svg>
</section>

<!-- ========== 第2层：可翻转的主体卡片 ========== -->
<section style="height: 0;">
  <svg viewBox="0 0 750 850" style="
    display: block;
    width: 100%;
    pointer-events: none;
  ">
    <!-- 容器缩放动画（翻转时整体缩小） -->
    <animate attributeName="width" 
             values="100%;100%;200%;200%" 
             dur="2s" 
             keyTimes="0;0.6;0.6;1"
             fill="freeze" 
             begin="mousedown"></animate>
    
    <animateTransform attributeName="transform" 
                      type="scale" 
                      values="1;1;0.5;0.5" 
                      dur="2s" 
                      keyTimes="0;0.6;0.6;1"
                      fill="freeze" 
                      begin="mousedown"></animateTransform>
    
    <!-- 松开时恢复 -->
    <animate attributeName="width" 
             values="200%;200%;100%;100%" 
             dur="2s" 
             keyTimes="0;0.5;0.5;1"
             fill="freeze" 
             begin="mouseup"></animate>
    
    <animateTransform attributeName="transform" 
                      type="scale" 
                      values="0.5;0.5;1;1" 
                      dur="2s" 
                      keyTimes="0;0.5;0.5;1"
                      fill="freeze" 
                      begin="mouseup"></animateTransform>
    
    <g>
      <!-- 【关键1】把旋转中心移到卡片中间（750/2=375） -->
      <g transform="translate(375 0)">
        <g>
          <!-- 【关键2】水平翻转动画：-1(镜像) → 1(正常) -->
          <animateTransform attributeName="transform" 
                            type="scale" 
                            values="-1 1;-1 1;1 1" 
                            dur="1.2s" 
                            keyTimes="0;0.166;1"
                            keySplines=".42,0,.58,1;.42,0,.58,1"
                            calcMode="spline"
                            fill="freeze" 
                            begin="mousedown"></animateTransform>
          
          <!-- 翻回去：1 → -1 -->
          <animateTransform attributeName="transform" 
                            type="scale" 
                            values="1 1;-1 1" 
                            dur="1s" 
                            keySplines=".42,0,.58,1"
                            calcMode="spline"
                            fill="freeze" 
                            begin="mouseup"></animateTransform>
          
          <!-- 【关键3】移回原点（抵消第一个 translate） -->
          <g transform="translate(-375 0)">
            
            <!-- ========== 背面内容 ========== -->
            <!-- 【重点】背面需要预先镜像存储 -->
            <g transform="translate(750 0) scale(-1 1)">
              <foreignObject x="85.5" y="15" width="579" height="820">
                <svg viewBox="0 0 579 820" style="
                  display: block;
                  width: 100%;
                  background: url('背面图片.png') 0 0 / 100% 100% no-repeat;
                "></svg>
              </foreignObject>
            </g>
            
            <!-- ========== 正面内容 ========== -->
            <g>
              <!-- 【关键4】正面的显示时机控制 -->
              <!-- 前70%时间隐藏，后30%显示 -->
              <animate attributeName="opacity" 
                       values="0;0;1;1" 
                       dur="1s" 
                       keyTimes="0;0.7;0.7;1"
                       fill="freeze" 
                       begin="mousedown"></animate>
              
              <!-- 松开时前50%显示，后50%隐藏 -->
              <animate attributeName="opacity" 
                       values="1;1;0;0" 
                       dur="1s" 
                       keyTimes="0;0.5;0.5;1"
                       fill="freeze" 
                       begin="mouseup"></animate>
              
              <foreignObject x="85.5" y="15" width="579" height="820">
                <svg viewBox="0 0 579 820" style="
                  display: block;
                  width: 100%;
                  background: url('正面图片.png') 0 0 / 100% 100% no-repeat;
                "></svg>
              </foreignObject>
            </g>
            
            <!-- ========== 点击接收层 ========== -->
            <g>
              <!-- 按下前移到屏幕外，按下后移回来 -->
              <animateTransform attributeName="transform" 
                                type="translate" 
                                values="-10000 0;-10000 0;0 0;0 0" 
                                dur="2s" 
                                keyTimes="0;0.6;0.6;1"
                                calcMode="discrete"
                                fill="freeze" 
                                begin="mousedown"></animateTransform>
              
              <animateTransform attributeName="transform" 
                                type="translate" 
                                values="0 0;0 0;-10000 0;-10000 0" 
                                dur="2s" 
                                keyTimes="0;0.5;0.5;1"
                                calcMode="discrete"
                                fill="freeze" 
                                begin="mouseup"></animateTransform>
              
              <!-- 透明但可点击的矩形 -->
              <rect x="0" y="0" width="750" height="850" 
                    fill="#000" 
                    opacity="0" 
                    style="pointer-events: visible;"></rect>
            </g>
            
          </g>
        </g>
      </g>
    </g>
  </svg>
</section>

<!-- ========== 第3层：最上层透明拦截（防误触） ========== -->
<section style="height: 0;">
  <svg viewBox="0 0 750 850" style="
    display: block;
    width: 100%;
    pointer-events: none;
  ">
    <g opacity="0">
      <rect x="0" y="0" width="750" height="850" 
            style="pointer-events: visible;" 
            fill="blue"></rect>
      
      <!-- 按下时移走，避免持续触发 -->
      <animateTransform attributeName="transform" 
                        type="translate" 
                        values="4000 0" 
                        dur="0.2s" 
                        calcMode="discrete"
                        begin="mousedown"></animateTransform>
    </g>
  </svg>
</section>

<!-- ========== 真正占高度的容器（撑开空间） ========== -->
<svg viewBox="0 0 750 850" style="
  display: block;
  width: 100%;
"></svg>
```

## 关键参数说明

| 参数 | 作用 | 调整方法 |
|------|------|----------|
| `viewBox="0 0 750 850"` | 画布尺寸 | 改成你的卡片实际尺寸 |
| `translate(375 0)` | 旋转中心点 | 改成 `宽度/2` |
| `scale(-1 1)` | 水平镜像 | 不要改 |
| `keyTimes="0;0.7;0.7;1"` | 正面显示时机 | 0.7 表示翻转70%时才显示 |
| `dur="1.2s"` | 翻转耗时 | 可改快/慢 |

## 正反面设计要点

### 背面（初始显示的那一面）：
```html
<!-- 必须用 scale(-1 1) 预先镜像 -->
<g transform="translate(750 0) scale(-1 1)">
  <foreignObject>
    <svg background="背面图片（设计时是正常方向的）"></svg>
  </foreignObject>
</g>
```

**为什么要镜像？**
- 整体 `scale` 从 `-1→1` 翻转时
- 背面会经过 `-1×(-1)=1`（正常）→ `0×(-1)=0`（压扁）→ `1×(-1)=-1`（镜像）
- 视觉效果 = 正常显示 → 压扁消失 → 翻到背后

### 正面（翻过来显示的那一面）：
```html
<!-- 不需要镜像，但要控制 opacity -->
<g>
  <animate attributeName="opacity" 
           values="0;0;1;1" 
           keyTimes="0;0.7;0.7;1"  ← 前70%隐藏
           begin="mousedown">
  </animate>
  
  <foreignObject>
    <svg background="正面图片（正常设计即可）"></svg>
  </foreignObject>
</g>
```

**为什么要延迟显示？**
- 前 70% 时间（0.84秒）：`scale` 从 `-1→0.4`，背面还在收窄
- 后 30% 时间（0.36秒）：`scale` 从 `0.4→1`，背面已压扁，正面安全显示

## 调试技巧

### 1. 看不到背面？
检查 `scale(-1 1)` 是否写在背面的 `<g>` 上

### 2. 翻转时重叠了？
调整 `keyTimes`，让正面更晚显示：
```html
keyTimes="0;0.8;0.8;1"  ← 改成80%时才显示
```

### 3. 点击没反应？
检查透明矩形是否有 `pointer-events: visible`[5]

### 4. 翻转方向反了？
把 `scale` 的 `values` 顺序反过来：
```html
<!-- 原本 -->
values="-1 1;1 1"

<!-- 改成 -->
values="1 1;-1 1"
```
